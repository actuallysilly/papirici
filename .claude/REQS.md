# Papirici

## What is this?

This project is a little game for two players
A boyfriend and a girlfriend

There's 3 kinds of papers (missions)
Blue (male), pink (female), white

Each player can draw their  won colour or white

The papers are little date-tasks

The players can add tasks and then draw them out
of the box

THE UI NEEDS TO BE STELLAR AND EXCELENTLY AND DYNAMICALLY ANIMATED WITH THREEJS

Papers are wrapped into basic wrappings
